### Manual Setup Instructions:
There are two ways to install the extension manually, cloning the repo or downloading the zipped ```.zip``` file from the GitHub Releases page.

To use any of these methods please 
1. Go to ```chrome://extensions/```
2. Turn ON "Developer Mode" (In Google Chrome; it's a toggle on the top right corner)

#### Using the Zip File (<span style="color: Green">Recommended</span>)
1. Enable the "Developer Mode"
2. [Download](https://github.com/akhanal47/replace-words-extn/releases) and save ```archive-web-page.zip``` file then unzip it to your desired location.
3. Select "Load Unpacked" button & select the folder where you unzipped the extension files.

#### Cloing the Repo
1. Clone the repo using ```git clone https://github.com/akhanal47/replace-words-extn```
1. Enable the "Developer Mode"
2. Select "Load Unpacked" button & select the folder where the repo is cloned
